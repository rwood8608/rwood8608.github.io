This is probably the worst homepage you'll ever see, if I'm completely honest with ya.

"Why is poorly organized code such a bad thing?" I hear you ask? Good question, I'll show you why.


Would you rather read code like this (NOTE: This is Pseudocode):

<HTML><HEADING><BODY><p>This code is a pain to read</p></BODY></HEADING></HTML>

Or code that looks like this?

<HTML>
<HEADING>
<BODY>
      <p>This code is easier to read!</p>
</BODY>
</HEADING>
</HTML>

Exactly my point. Although to be fair, a lot of the code for this site is pretty organized to me, it probably isn't to anyone else
to anyone else, due to my terrible definition of the word "organization".
